My First Freelance Analytics Project
How I delivered two real world data projects under tight deadlines – and what I learned along the way.

# “I don’t need more visuals – I need the right insights.”
That was my turning point. After hours of debugging relationships and wrestling with regression outputs, I learned that adding more charts doesn’t fix a broken analysis. What matters is asking the right questions, verifying every number, and knowing when to simplify.

This project was my first real world freelance engagement as a data analyst. It challenged my technical skills, my patience, and my ability to communicate value under tight deadlines. Below is the story of how I turned ambiguous requirements and messy datasets into two professional deliverables: a Power BI dashboard and a predictive analytics report.
________________________________________

# Project Background
A logistics company wanted to evaluate the performance of two distribution centres – not just revenue, but cost efficiency, shipment attainment, and operational KPIs. Separately, a digital economy study required identifying which youth entrepreneurial capabilities (platform engagement, digital skills, monetisation activity) best predict monthly income.
My task: Deliver an interactive dashboard (Power BI) + 3 minute video presentation, and a 4,000 word regression report – all in #1_day.

•	Microsoft Excel (multiple regression, data validation)

# Deliverables & Key Outcomes
Power BI Dashboard
•	Visuals: 4 different chart types (KPI cards, clustered column, bar, line+column combo) – exceeded the required 3.
•	Insights:
o	Both centres are profitable, but Penampang has a stronger margin (24.9% vs 22.7%).
o	Shipment attainment is below target in all categories (Inanam ~90%, Penampang ~86%).
o	Operational metrics (on time delivery, workforce productivity) need improvement at both centres.
•	Video presentation: Delivered in exactly 3 minutes, using a script I rehearsed to stay concise and confident.

Regression Report (SOAR Analytics Model)
•	Prediction equation: INC = 4.65 + 0.050(ENG) + 0.088(SKL) + 0.509(MON)
•	Key finding: Monetisation activity is the strongest predictor – each additional monetised activity adds ~RM50.92 to monthly income. Digital skills also matter (RM8.81 per training hour), but raw platform engagement alone is not significant.
•	Actionable recommendations:
1.	Youth entrepreneurs should prioritise weekly monetisation calendars.
2.	Platform providers (e.g., TikTok) should build learning paths linking skills to monetisation tools.
3.	Policymakers must design training that combines practice and income measurement.
4.	Support should be segmented by activity level – beginners need different help than advanced sellers.
________________________________________

# What This Project Taught Me (Beyond Tools)
•	Data validation is non negotiable. One wrong row in regression can flip a conclusion.
•	Communication > complexity. A simple profit chart with a clear table beats an over engineered combo chart.
•	Know when to stop adding. More visuals do not impress – clear answers do.
•	Freelancing is problem solving first, coding second. Every hour spent on a relationship error was an hour not spent on insights. I learned to ask “Do I need this?” before building it.
